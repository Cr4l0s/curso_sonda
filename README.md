- Múltiplos de 2 hasta 500
- Juego "Vanilla Ice" (divisible por 5 y 10)

- Función `iterarDiccionario2(llave, lista)`
- Función `imprimirInformacion(diccionario)`

### 3. Análisis de Ventas (`analisis_ventas_1.py`)
- Carga de datos de ventas
- Cálculo de ingresos totales
- Producto más vendido
- Precio promedio por producto
- Ingresos por día
- Resumen con diccionario anidado
